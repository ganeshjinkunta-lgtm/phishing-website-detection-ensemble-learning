# Phishing Website Detection Using Ensemble Learning

## Overview
This project implements a Boosting-Based Hybrid Feature Selection and Multi-Layer Stacked Ensemble Learning Model for phishing website detection. The system analyzes website-related features and predicts whether a website is legitimate or phishing.

## Features
- Phishing website detection
- Hybrid feature selection
- Multi-layer stacked ensemble learning
- Dataset analysis and prediction
- User-friendly web interface

## Technologies Used
- Python
- Django
- Machine Learning
- HTML
- CSS
- CSV Dataset

## Project Structure
- Remote_User Module
- Service_Provider Module
- Dataset Management
- Prediction and Analysis Module

## Dataset
The project uses a phishing website dataset containing multiple website features for classification.

## Results
The proposed model improves phishing website detection accuracy and reduces false positive rates compared to traditional approaches.
